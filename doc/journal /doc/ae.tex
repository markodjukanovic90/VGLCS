	\section*{Abstract}
	
 	The Variable Gapped Longest Common Subsequence Problem (VGLCSP) is a challenging combinatorial optimization problem arising in sequence analysis with applications in computational biology and time series analysis, where the search space grows rapidly with the number and with gap constraints. Beam Search--based heuristics can provide good approximate solutions, but their effectiveness strongly depends on the quality of the heuristic used to evaluate partial subsequences.
 	
 	This work proposes a learning-based framework to automatically construct a heuristic for guiding Beam Search (BS) in the VGLCSP. The approach integrates a feed-forward neural network (FFNN) within the Iterative Multi-Source BS (IMSBS) algorithm, where the network estimates the quality of candidate extensions during the search. Instead of gradient-based training, the network parameters are optimized using a Genetic Algorithm (GA), allowing the heuristic to be learned directly from solution quality obtained during the search process. Compact feature representations are designed to encode the state of partial solutions, enabling the learned heuristic to generalize across instances with varying sequence lengths, numbers of sequences, and alphabet sizes. During training, candidate neural networks are evaluated by embedding them within the BS procedure and measuring solution quality on a set of training instances, while separate validation instances are used to monitor generalization.
 	
 	Preliminary experiments indicate that the learned heuristic can improve the performance of the baseline IMSBS algorithm on a range of VGLCSP instances. These results demonstrate the potential of combining evolutionary optimization with neural network models to automatically design effective heuristics for complex combinatorial search problems.

	\section{Introduction}
	
	The Longest Common Subsequence (LCS) problem and its variants are fundamental problems in sequence analysis and bioinformatics. 
	One particularly challenging variant is the \textit{Variable Gapped Longest Common Subsequence Problem} (VGLCSP)~\cite{penga2011longest}, where gap constraints restrict the allowable distances between consecutive matches of solutions in each input sequence. 
	These constraints significantly increase the complexity of the search space and make exact algorithms impractical for medium and large instances.
	
	Heuristic search methods such as Beam Search (BS) have been proposed to handle this complexity by exploring only the most promising candidate solutions. 
	However, the effectiveness of BS strongly depends on the quality of the heuristic used to rank partial solutions.	Traditional heuristics are typically handcrafted and remain static across instances, limiting their ability to adapt to diverse problem structures~\cite{djukanovic_eurocast_2026}.
	
	This work introduces a learning-based framework for automatically constructing such heuristics. 
	The key idea is to integrate machine learning with combinatorial search so that the heuristic function is learned directly from data and optimized according to the final solution quality achieved by the search algorithm.

	\section{Proposed Approach}
	
	The proposed framework combines three main components:
	
	\begin{enumerate}
		\item \textbf{Iterative Multi-Source Beam Search (IMSBS)}~\cite{djukanovic_eurocast_2026} for constructing candidate subsequences.
		\item \textbf{Feed-Forward Neural Network (FFNN)}~\cite{svozil1997introduction} acting as a heuristic evaluation function for partial solutions.
		\item \textbf{Genetic Algorithm (GA)}~\cite{gonccalves2011biased} for optimizing the neural network parameters.
	\end{enumerate}
	
	In the learning process, each GA individual represents a flattened vector of neural network weights. 
	For each individual, the weights are loaded into the FFNN, which evaluates partial states generated during a BS execution. 
	The IMSBS algorithm then constructs subsequences guided by the neural heuristic. The fitness of each individual corresponds to the average solution quality obtained on a training set of instances. 
	Validation instances are evaluated separately to monitor generalization performance.
	
	A key aspect of the approach is the design of compact feature representations for partial search states. 
	Each node in the search tree is described by a feature vector containing:
	
	\begin{itemize}
		\item normalized positions of current matches in each sequence
		\begin{itemize}
			\item   statistical descriptors (minimum, maximum, mean, and standard deviation) of these positions;
	    \end{itemize}
		\item the current length of the partial subsequence,
		\item statistical summaries of remaining gap constraints,
		\item global instance features such as alphabet size $|\Sigma|$.
	\end{itemize}
	Some other global features have been considered such as number of sequences ($m$) or lenght of input sequences ($n$), however without improving the results.
	
	These features provide a low-dimensional yet informative representation of the search state and allow the neural network to generalize across instances of different sizes. The learning process is illustrated in Figure~\ref{fig:ga_imsbs}.

	\section{Experimental Results}
	
	This section includes datasets with preliminary experiments described in the paper. 
	The experiments evaluate the performance of the learning-based Beam Search (LIMSBS) against the baseline IMSBS algorithm proposed in~\cite{djukanovic_eurocast_2026}.
	
	A set of random instances is generated in the letter paper, where for each combinations  number of sequences $m \in \{2,3,5,10\}$, sequence length $n \in \{50,100,200,500\}$, and alphabet size $|\Sigma| \in \{2,4\}$, 10 random instances is generated. All sequences are randomly generated. In overall, 320 problem instances are provided. Both algorithms are tuned by the grid search. The best-performing neural network configuration uses three hidden layers with 10 neurons each and the \texttt{ReLU} activation function, among the three tested (\texttt{ReLU}, \texttt{sigmoid}, and \texttt{tanh}).	Training of the neural heuristic using BRKGA is performed for two hours on the High performance computing infrastructure VEGA, placed at IZUM in Maribor, Slovenia. 
	
	Despite this relatively modest training budget, the learned heuristic demonstrates competitive performance and improves average solution quality in several instance categories, particularly for larger numbers of sequences, see numerical results in Table~\ref{tab:preliminary_results}.  These results indicate that learning-based heuristics can effectively guide BS and adapt to structural characteristics of the sequence alignment problems, in this case the VGLCS, especially suited for $m\geq 5$.

	\section{Conclusions}
	
	This artifact provides the rough implementation details and experimental framework for learning heuristics in combinatorial sequence optimization using neuro-evolutionary methods. 
	By integrating Beam Search with neural networks optimized via genetic algorithms, the approach enables adaptive heuristic construction in non-differentiable settings.
	
	The   reported results provide a promising basis for further experimentation on learning-guided combinatorial search methods. 
	Beyond the VGLCSP, the proposed framework can be extended to other sequence alignment and combinatorial optimization problems where heuristic design is difficult.
