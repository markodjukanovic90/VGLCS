	\section*{Abstract}
	
 	The Variable Gapped Longest Common Subsequence Problem (VGLCSP) is a challenging combinatorial optimization problem arising in sequence analysis, where the search space grows rapidly with the number and length of sequences and with gap constraints. Beam Search–based heuristics can provide good approximate solutions, but their effectiveness strongly depends on the quality of the heuristic used to evaluate partial subsequences.
 	
 	This work proposes a learning-based framework to automatically construct a heuristic for guiding Beam Search in the VGLCSP. The approach integrates a feed-forward neural network (FFNN) within the Iterative Multi-Source Beam Search (IMSBS) algorithm, where the network estimates the quality of candidate extensions during the search. Instead of gradient-based training, the network parameters are optimized using a Genetic Algorithm (GA), allowing the heuristic to be learned directly from solution quality obtained during the search process.
 	
 	Compact feature representations are designed to encode the state of partial solutions, enabling the learned heuristic to generalize across instances with varying sequence lengths, numbers of sequences, and alphabet sizes. During training, candidate neural networks are evaluated by embedding them within the Beam Search procedure and measuring solution quality on a set of training instances, while separate validation instances are used to monitor generalization.
 	
 	Preliminary experiments indicate that the learned heuristic can improve the performance of the baseline IMSBS algorithm on a range of VGLCSP instances. These results demonstrate the potential of combining evolutionary optimization with neural network models to automatically design effective heuristics for complex combinatorial search problems.

	\section{Introduction}
	
	The Longest Common Subsequence (LCS) problem and its variants are fundamental problems in sequence analysis and bioinformatics. 
	One particularly challenging variant is the \textit{Variable Gapped Longest Common Subsequence Problem} (VGLCSP), where gap constraints restrict the allowable distances between consecutive matches in each sequence. 
	These constraints significantly increase the complexity of the search space and make exact algorithms impractical for medium and large instances.
	
	Heuristic search methods such as Beam Search have been proposed to handle this complexity by exploring only the most promising candidate solutions. 
	However, the effectiveness of Beam Search strongly depends on the quality of the heuristic used to rank partial solutions. 
	Traditional heuristics are typically handcrafted and remain static across instances, limiting their ability to adapt to diverse problem structures.
	
	This work introduces a learning-based framework for automatically constructing such heuristics. 
	The key idea is to integrate machine learning with combinatorial search so that the heuristic function is learned directly from data and optimized according to the final solution quality achieved by the search algorithm.

	\section{Proposed Approach}
	
	The proposed framework combines three main components:
	
	\begin{enumerate}
		\item \textbf{Iterative Multi-Source Beam Search (IMSBS)} for constructing candidate subsequences.
		\item \textbf{Feed-Forward Neural Network (FFNN)} acting as a heuristic evaluation function for partial solutions.
		\item \textbf{Genetic Algorithm (GA)} for optimizing the neural network parameters.
	\end{enumerate}
	
	In the learning process, each GA individual represents a flattened vector of neural network weights. 
	For each individual, the weights are loaded into the FFNN, which evaluates partial states generated during Beam Search. 
	The IMSBS algorithm then constructs subsequences guided by the neural heuristic.
	
	The fitness of each individual corresponds to the average solution quality obtained on a training set of instances. 
	Validation instances are evaluated separately to monitor generalization performance.
	
	A key aspect of the approach is the design of compact feature representations for partial search states. 
	Each node in the search tree is described by a feature vector containing:
	
	\begin{itemize}
		\item normalized positions of current matches in each sequence,
		\item statistical descriptors (minimum, maximum, mean, and standard deviation) of these positions,
		\item the current length of the partial subsequence,
		\item statistical summaries of remaining gap constraints,
		\item global instance features such as alphabet size $|\Sigma|$ and number of sequences $m$.
	\end{itemize}
	
	These features provide a low-dimensional yet informative representation of the search state and allow the neural network to generalize across instances of different sizes.

	\section{Experimental Results}
	
	The artifact includes scripts and datasets enabling reproduction of the preliminary experiments described in the paper. 
	The experiments evaluate the performance of the learning-based Beam Search (LIMSBS) against the baseline IMSBS algorithm.
	
	Instances vary in the number of sequences $m \in \{2,3,5,10\}$, sequence length $n \in \{50,100,200,500\}$, and alphabet size $|\Sigma| \in \{2,4\}$. 
	The best-performing neural network configuration uses three hidden layers with 10 neurons each and the ReLU activation function.
	
	Training of the neural heuristic using BRKGA is performed for two hours. 
	Despite this relatively modest training budget, the learned heuristic demonstrates competitive performance and improves average solution quality in several instance categories, particularly for larger numbers of sequences.
	
	These results indicate that learning-based heuristics can effectively guide Beam Search and adapt to structural characteristics of sequence alignment problems.

	\section{Conclusions}
	
	This artifact provides the implementation and experimental framework for learning heuristics in combinatorial sequence optimization using neuro-evolutionary methods. 
	By integrating Beam Search with neural networks optimized via genetic algorithms, the approach enables adaptive heuristic construction in non-differentiable settings.
	
	The provided code and datasets allow reproduction of the reported results and offer a basis for further experimentation on learning-guided combinatorial search methods. 
	Beyond the VGLCSP, the proposed framework can be extended to other sequence alignment and combinatorial optimization problems where heuristic design is difficult.
