	\section*{Abstract}
		 
 	The Variable Gapped Longest Common Subsequence Problem (VGLCSP) is a challenging combinatorial optimization problem arising in sequence analysis with applications in computational biology and time series analysis, where the search space grows rapidly with the number of input and gap constraints. Beam Search--based heuristics can provide good approximate solutions here, but their effectiveness strongly depends on the quality of the heuristic used to evaluate partial solutions.
 	
 	This work proposes a learning-based framework to automatically generate a heuristic for guiding Beam Search (BS) in the VGLCSP. The approach integrates a feed-forward neural network (FFNN) within the Iterative Multi-Source BS (IMSBS), where the network estimates the quality of candidate extensions during the search. Instead of gradient-based training, the network parameters are optimized using a Genetic Algorithm (GA), allowing the heuristic to be learned directly from the quality of solutions obtained during the search process. Compact feature representations are designed to encode the state of partial solutions, enabling the learned heuristic to generalize across instances with varying sequence lengths, numbers of sequences, and alphabet sizes. During training, candidate neural networks are evaluated by embedding them within the BS procedure and measuring solution quality on a set of training instances, while separate validation instances are used to monitor generalization.
 	
 	Preliminary experiments indicate the learned heuristic can improve the performance of the baseline IMSBS on a range of VGLCSP instances. They demonstrate the potential of combining evolutionary strategies with NN models to automatically design effective heuristics for complex combinatorial  problems.

	\section{Introduction}
	
	The Longest Common Subsequence (LCS) problem and its variants are fundamental problems in sequence analysis and bioinformatics. 
	One particularly challenging variant is the \textit{Variable Gapped Longest Common Subsequence Problem} (VGLCSP)~\cite{penga2011longest}, where gap constraints restrict the allowable distances between consecutive matches of solutions in each input sequence. 
	These constraints significantly increase the complexity of the search space and make exact algorithms impractical for medium and large instances.
	
	Heuristic search methods such as Beam Search (BS) have been proposed to handle this complexity by exploring only the most promising candidate solutions. 
	However, the effectiveness of BS strongly depends on the quality of the heuristic used to rank partial solutions.	Traditional heuristics are typically handcrafted and remain static across instances, limiting their ability to adapt to diverse problem structures~\cite{djukanovic_eurocast_2026}.
	
	This work introduces a learning-based framework for automatically constructing such heuristics. 
	The key idea is to integrate machine learning with combinatorial search so that the heuristic function is learned directly from data and optimized according to the final solution quality achieved by the search algorithm.

	\section{Proposed Approach}
	
	The proposed framework combines three main components:
	
	\begin{enumerate}
		\item \textbf{Iterative Multi-Source Beam Search (IMSBS)}~\cite{djukanovic_eurocast_2026} for constructing feasible solutions of VGLCSP.
		\item \textbf{Feed-Forward Neural Network (FFNN)}~\cite{svozil1997introduction} acting as a heuristic evaluation function for partial solutions.
		\item \textbf{Genetic Algorithm (GA)}~\cite{gonccalves2011biased} for optimizing NN parameters.
	\end{enumerate}
	
	In the learning process, each GA individual represents a flattened vector of neural network parameters. 
	For each individual, the parameters are loaded into the FFNN, which evaluates partial solutions generated during a BS execution. 
	The IMSBS algorithm then constructs complete solutions guided by the neural heuristic. The fitness of each individual corresponds to the average solution quality obtained on a set of training instances. 
	Validation instances are evaluated separately to monitor generalization performance.
	
	A key aspect of the approach is the design of compact feature representations for partial solutions. 
	Each node in the search graph is described by a feature vector containing
		(i) normalized positions of current matches in each sequence:
	     statistical descriptors (minimum, maximum, mean, and standard deviation) of these positions; (ii)  the current length of the partial solutions; (iii) statistical summaries of remaining gap constraints; (iv) global instance features such as alphabet size $|\Sigma|$. Some other global features have been taken into account such as the number of sequences ($m$) or the  length of input sequences ($n$), however without a major effect on the final results.	These features provide a low-dimensional yet informative representation of the search states and allow the network to generalize across instances of different sizes. 	\section{Experimental Results}
	

	The experiments evaluate the performance of the learning-based Beam Search (LIMSBS) against the baseline IMSBS algorithm proposed in~\cite{djukanovic_eurocast_2026} on the set of random instances generated in the latter paper. For each combination of number of sequences $m \in \{2,3,5,10\}$, sequence length $n \in \{50,100,200,500\}$, and alphabet size $|\Sigma| \in \{2,4\}$, 10 random instances are generated.  Overall, 320 problem instances are provided. Both algorithms are tuned by grid search. The best-performing NN configuration uses three hidden layers with 10 neurons each and the \texttt{ReLU} activation function, among the three tested (\texttt{ReLU}, \texttt{sigmoid}, and \texttt{tanh}).	FFNN training is performed with two hours time limit on the High performance computing infrastructure VEGA, placed at IZUM in Maribor, Slovenia, equipped with AMD EPYC 7H12 CPUs operating
	at 2.35 GHz.   Other crucial parameters in both  IMSBS and LIMSBS are set to beam width $\beta=1000$, allowing 1000 BS iterations.
	
	Despite this relatively modest training budget, the learned heuristic demonstrates competitive performance and improves average solution quality in several instance categories, particularly for larger numbers of sequences, see numerical results in Table~\ref{tab:preliminary_results}.  These results indicate learning-based heuristics can effectively guide BS and adapt to structural characteristics of the sequence  problems, in this case the VGLCSP, especially suited for $m\geq 5$. The average runtimes of both approaches are comparable.

	\section{Conclusions}
	
	This poster paper provides the principal implementation details and the experimental framework for learning heuristics in combinatorial sequence optimization using neuro-evolutionary methods. 
	By integrating BS with neural networks optimized via genetic algorithms, the approach enables adaptive heuristic generation in non-differentiable settings. The   reported results provide a promising basis for further experimentation on learning-guided combinatorial search methods. 
	Beyond the VGLCSP, the proposed framework can be extended to other sequence alignment and combinatorial optimization problems where heuristic design is difficult.
